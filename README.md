# Voituree
Landing Page for a car selling and leasing company created with React Js and SCSS.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

### Getting Started
1. Clone this repo
2. Run ```npm install```
3. Run ```npm start ```
